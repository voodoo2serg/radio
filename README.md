# Архив Telegram-каналов российских радиостанций

Открытый исследовательский корпус публикаций официальных Telegram-каналов российских радиостанций. Репозиторий создан для воспроизводимого продольного анализа цифрового присутствия, платформенной инфраструктуры и самостоятельности медианарратива по отношению к линейному эфиру.

## Состав корпуса

| Каталог | Радиостанция | Основной период непрерывного анализа |
|---|---|---|
| `BFM/` | Business FM | 2017–2026H1 |
| `Mayak/` | «Маяк» | 2022–2026H1 |
| `Radio1/` | «Радио 1» | 2019–2026H1 |
| `Dorozhnoe/` | «Дорожное радио» | 2024–2026H1 |
| `EuropePlus/` | «Европа Плюс» | дополнительный архив; не включён в основное сравнение |

Каждый каталог содержит HTML-файлы экспорта Telegram (`messages.html`, `messages2.html` и далее). Файлы необходимо анализировать только внутри каталогов станций.

## Производные данные

Каталог `data/` содержит обезличенные результаты кодирования:

- `coded_sample.csv` — выборка для годовых траекторий;
- `profile_summary.csv` — агрегированные годовые профили;
- `comparison_coded_sample.csv` — отдельная сравнительная выборка;
- `comparison_2024H1_2026H1.csv` — сопоставление первых полугодий 2024 и 2026 гг.

Определения переменных, правила отбора и кодирования приведены в [CODEBOOK.md](CODEBOOK.md).

## Ограничения

Корпус отражает только публично доступные сообщения и инфраструктурные связи, предъявленные через Telegram. Отсутствие ссылки не доказывает отсутствие площадки или редакционной связи. Временные лакуны перечислены в кодбуке.
